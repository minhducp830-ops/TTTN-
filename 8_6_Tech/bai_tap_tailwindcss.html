📚 Bài Tập 3: Tổng Hợp Các Class TailwindCSS Đã Dùng

Liệt kê và giải thích các utility class quan trọng đã sử dụng trong bài.

📐 Layout

  • flex → Bật chế độ Flexbox
  • grid → Bật chế độ Grid
  • grid-cols-3 → Chia 3 cột bằng nhau
  • gap-6 → Khoảng cách giữa các item
  • max-w-4xl mx-auto → Căn giữa, giới hạn rộng

📏 Spacing

  • p-6 → Padding tất cả chiều 1.5rem
  • px-4 py-3 → Padding ngang/dọc riêng
  • mb-6 → Margin bottom 1.5rem
  • space-y-5 → Khoảng cách dọc các con
  • w-full → Chiều rộng 100%

✍️ Typography

  • text-3xl font-bold → Cỡ chữ to, đậm
  • text-sm → Chữ nhỏ
  • text-gray-400 → Màu chữ xám nhạt
  • text-center → Căn giữa văn bản
  • text-primary → Màu chủ đạo (đỏ)

✨ Hiệu ứng

  • rounded-xl → Bo góc lớn
  • shadow-lg → Đổ bóng lớn
  • hover:border-primary → Đổi border khi hover
  • transition-all duration-300 → Animation mượt 300ms
  • focus:ring-1 focus:ring-primary → Viền khi focus
